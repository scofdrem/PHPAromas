## Безопасность
- Никогда не хардкодь секреты, API-ключи, пароли
- Используй переменные окружения (.env) для конфиденциальных данных
- Всегда валидируй пользовательский ввод
- Используй параметризированные запросы для SQL (защита от SQL-инъекций)
- Экранируй HTML-вывод (защита от XSS)
- Используй HTTPS для всех API-вызовов
- Добавляй Rate Limiting для публичных эндпоинтов
